# ウェイトリストLP ビジュアル強化 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** `/lp` を「温かいフレンドリー」トーンで魅力的にし、山分けフロー図インフォグラフィックを主役に据える。

**Architecture:** 既存の `src/features/waitlist-lp/` コンポーネントをTailwindユーティリティのみで装飾強化し、新規の `YamawakeFlow`（固定金額例のフロー図）と共有 `SectionCard`（アイコンチップ付きカード）を追加する。グローバルCSS・テーマ・コピー・ロジックは変更しない。

**Tech Stack:** React 19 (Server Components) / Tailwind CSS 4 / lucide-react（導入済み）/ Vitest + Testing Library

**Spec:** `docs/superpowers/specs/2026-07-05-lp-visual-design.md`

## Global Constraints

- **既存のテキストコピーを一切変更しない**（見出し・本文・ラベル・ボタン・メッセージ）。既存テスト・E2Eが無修正で通ることが完了条件。新規テキストの追加（バッジ「先行登録で6ヶ月無料」、フロー図の見出し・ラベル）は可
- `globals.css` / テーマ変数 / LP以外のファイルを変更しない。新規npm依存・画像ファイル・外部フォントを追加しない
- ダークモードは `dark:` バリアントで両対応
- フロー図の担当者カラーは検証済みパレットを使用: **あなた = rose-600 `#e11d48`（Tailwind `bg-rose-600`）/ パートナー = amber-700 `#b45309`（`bg-amber-700`）**（datavizバリデーターでライト/ダーク両モード全チェックPASS済み。変更する場合は再検証が必要）。経費ブロックは意図的なニュートラル（`bg-stone-400 dark:bg-stone-600`）
- フロー図のマーク仕様: セグメント間ギャップ2px（`gap-0.5`）、データ端の角丸4px、全セグメントに直接ラベル（色だけに依存しない）
- named export のみ。コミットはConventional Commits形式・日本語、生成情報なし
- テスト実行は `npx vitest run <path>`（watchモード禁止）

```tsx
interface FlowStepProps {
  number: number
  title: string
  children: React.ReactNode
}

function FlowStep({ number, title, children }: FlowStepProps) {
  return (
    <div>
      <p className="flex items-center gap-2 text-sm font-semibold">
        <span
          className="inline-flex size-6 items-center justify-center rounded-full bg-gradient-to-br from-rose-500 to-amber-500 text-xs font-bold text-white"
          aria-hidden="true"
        >
          {number}
        </span>
        {title}
      </p>
      <div className="mt-3">{children}</div>
    </div>
  )
}
```

```tsx
import type { LucideIcon } from 'lucide-react'

interface SectionCardProps {
  icon: LucideIcon
  title: string
  description: string
  layout?: 'stack' | 'row'
}

export function SectionCard({
  icon: Icon,
  title,
  description,
  layout = 'stack',
}: SectionCardProps) {
  const chip = (
    <span
      className="inline-flex size-10 shrink-0 items-center justify-center rounded-full bg-gradient-to-br from-amber-100 to-rose-100 text-rose-600 dark:from-amber-900/40 dark:to-rose-900/40 dark:text-rose-300"
      aria-hidden="true"
    >
      <Icon className="size-5" />
    </span>
  )

  if (layout === 'row') {
    return (
      <div className="flex items-start gap-4 rounded-2xl border bg-background p-6 shadow-sm">
        {chip}
        <div>
          <h3 className="font-semibold">{title}</h3>
          <p className="mt-2 text-sm text-muted-foreground">{description}</p>
        </div>
      </div>
    )
  }

  return (
    <div className="rounded-2xl border bg-background p-6 shadow-sm">
      {chip}
      <h3 className="mt-4 font-semibold">{title}</h3>
      <p className="mt-2 text-sm text-muted-foreground">{description}</p>
    </div>
  )
}
```
